# TradeBlog 博客系统实现计划

## 技术栈
- **前端**: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **数据库**: SQLite + better-sqlite3
- **富文本编辑器**: Tiptap
- **瀑布流**: CSS Columns + React
- **后台保护**: 简单密码或环境变量

## 核心功能模块

### 1. 数据库设计
- 文章表（标题、内容、摘要、封面、发布时间等）
- 分类表、标签表
- 统计表（阅读量）
- API 密钥表（机器人认证）
- 后台管理员配置表

### 2. 权限系统
- **游客**：只能阅读文章（瀑布流、详情页、搜索）
- **管理员**：通过简单密码登录后台，管理文章
- **机器人**：通过 API Key 操作文章（增删改查）

### 3. 后台管理系统（简单密码保护）
- 文章管理（CRUD、富文本编辑器）
- 分类和标签管理
- 统计数据（访问量、热门文章）
- API 密钥管理（生成、撤销）
- 简单密码登录

### 4. 对外 API（机器人对接）
- POST /api/articles - 创建文章（需 API Key）
- PUT /api/articles/:id - 更新文章（需 API Key）
- DELETE /api/articles/:id - 删除文章（需 API Key）
- GET /api/articles - 获取文章列表（公开）
- GET /api/articles/:id - 获取文章详情（公开）
- GET /api/articles/search - 搜索文章（公开）
- 使用 API Key 认证（Header: X-API-Key）

### 5. 前端功能（游客访问）
- **瀑布流主页**：
  - 无限滚动加载
  - 显示文章摘要、封面、标签、发布时间
  - 响应式布局
- **文章详情页**：
  - 默认显示摘要
  - 点击"展开阅读"显示完整内容
  - 支持"收起"折叠
  - 显示阅读量
  - 相关文章推荐
- **搜索和筛选**：
  - 关键词搜索
  - 标签筛选
  - 分类筛选
  - 日期归档
- **其他页面**：
  - 标签页
  - 分类页
  - 归档页

### 6. UI 设计
- 响应式布局（移动端适配）
- 自定义 LOGO（SVG）
- 暗色/亮色主题切换
- 加载动画和骨架屏
- 平滑滚动和过渡动画
